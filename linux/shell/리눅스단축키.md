리눅스 첫 설치하면 설정 가서 둘러보는게 좋다.

`ctrl + alt + T` : 터미널
`$ sudo reboot` : 재부팅 *재부팅을 요구하는 작업의 경우 shutdown이 아닌 reboot를 해야함*
`$ shutdown -h now` : 종료
`$ sudo halt -p` : 전원을 내림. 우분투 기준으로는 `$ shutdown -h now`와 같음 (전원을 내림, 우분투 한정으로 안전한 명령어)
`$ sudo apt update`
`$ sudo apt upgrade`
`$ sudo apt autoremove`
`$ sudo apt autoclean`
